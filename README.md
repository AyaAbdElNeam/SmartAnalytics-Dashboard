Excel Sales Data Analysis & Interactive Dashboard

Project Overview
This project analyzes a multi-country sales dataset and presents insights through a fully interactive dashboard built using **Microsoft Excel**. The analysis covers sales performance across regions, countries, customer segments, product categories, and shipping modes to identify trends and profitability patterns.

Tools & Techniques
- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Data Cleaning & Data Transformation  
- Conditional Formatting  
- KPI Development  
- Interactive Slicers & Filters  

📊 Dataset Description
The dataset includes:
- Order details (Order ID, Order Date, Shipping Mode)
- Customer information (City, Country, Region, Segment)
- Product data (Category, Sub-Category, Product Name)
- Sales metrics (Unit Price, Quantity, Discount, Profit)

Key Features
- Sales performance analysis by **Region and Country**
- Profitability insights by **Category and Sub-Category**
- Segment-based performance comparison (Consumer, Corporate, Home Office)
- Interactive dashboard with dynamic filtering
- KPI tracking for overall performance monitoring

Business Insights
The dashboard enables users to:
- Identify top-performing cities and regions
- Analyze the impact of discounts on profit
- Compare sales distribution across customer segments
- Detect trends in shipping modes and product categories

How to Use
1. Download the Excel file from this repository.
2. Open it using Microsoft Excel (Desktop recommended).
3. Navigate to the Dashboard sheet.
4. Use slicers to filter by region, segment, or category.
5. Explore insights through interactive charts and KPIs.
