# 📊 Excel Data Analysis & Interactive Dashboard

## 📌 Project Overview
This project presents a complete data analysis process and an interactive dashboard built using **Microsoft Excel**. The goal is to transform raw data into meaningful insights through visualization and structured reporting.

## 🛠 Tools & Techniques
- Microsoft Excel  
- Pivot Tables  
- Charts & Data Visualization  
- Conditional Formatting  
- Data Cleaning & Transformation  

## 🚀 Features
- Interactive dashboard with filters and slicers  
- Dynamic charts and visual reports  
- Key Performance Indicators (KPIs)  
- Clean and professional layout  
- Easy data exploration and trend analysis  

## 📂 How to Use
1. Download the Excel file from this repository.
2. Open it using Microsoft Excel.
3. Navigate between dashboard sheets.
4. Use filters and slicers to explore insights.

## 📈 Project Purpose
This project demonstrates practical skills in data analysis, visualization, and dashboard design using Excel to support data-driven decision making.
